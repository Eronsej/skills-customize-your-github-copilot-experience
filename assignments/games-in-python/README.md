
# 🎮 Hangman Game Challenge

## 🎯 Objective

Build a playable Hangman game in Python that demonstrates string manipulation, control flow, and basic I/O. Focus on clear code structure and user-friendly terminal interaction.

## 🔧 Prerequisites

- Python 3.8+
- Basic knowledge of strings, loops, and conditionals

## 📁 Files Provided

- starter-code.py — starter template for the game

## 📝 Tasks

### 🛠️ Game Logic

#### Description
Implement the core mechanics: word selection, tracking guessed letters, revealing progress, and managing remaining attempts.

#### Requirements
- Randomly select a word from a predefined list
- Accept single-letter guesses and validate input
- Display the current word state (e.g., _ a _ _ n)
- Track and display remaining incorrect attempts
- End the game with win or lose message

### 🛠️ User Interface & Extras

#### Description
Improve player experience and add optional features for extra credit.

#### Requirements
- Show guessed letters and number of attempts left
- Clear, labeled prompts and messages
- (Optional) Add difficulty levels, word categories, or a scoring system

## ✅ Deliverables

- Completed game script (updated starter-code.py or game.py)
- Optional: extras implemented noted in a short README section

## ▶️ How to run

1. (Optional) Create a virtual env and install dependencies (none required for standard library):

   python -m venv .venv && source .venv/bin/activate

2. Run the game:

   python starter-code.py

3. Play in the terminal following on-screen prompts
